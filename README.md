# Parkinson's Disease Prediction System

A Machine Learning-based system to predict Parkinson’s disease using biomedical voice measurements. The model analyzes input features and provides real-time predictions with confidence scores.

## Features

* Predicts Parkinson’s disease using ML models
* Achieved **94.8% accuracy** using Random Forest
* Evaluates multiple models (Logistic Regression, SVM, Decision Tree, KNN)
* Uses feature scaling and correlation-based feature selection
* Real-time prediction through Streamlit web app

## Tech Stack

* Python
* NumPy, Pandas
* Scikit-learn
* Matplotlib, Seaborn
* Streamlit

## Machine Learning Workflow

1. Data preprocessing (cleaning, feature selection)
2. Feature scaling using StandardScaler
3. Model training and comparison
4. Evaluation using Accuracy, Recall, ROC-AUC
5. Deployment using Streamlit
